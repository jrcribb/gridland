The `Link` component from `@gridland/ui` renders a clickable hyperlink in the
terminal UI.

<div className="my-8">
  <LinkDemo />
</div>

<RunDemo name="link" />

## Installation

```bash
bunx shadcn@latest add https://raw.githubusercontent.com/cjroth/gridland/main/packages/ui/registry.json link
```

## Usage

```tsx title="Example"
import { Link } from "@/components/ui/link"

<Link url="https://opentui.com">Visit opentui.com</Link>
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `children` | `ReactNode` | Link text content |
| `url` | `string` | Target URL |
| `underline` | `"solid" \| "dashed" \| "dotted" \| "none"` | Underline style (default `"solid"`) |
