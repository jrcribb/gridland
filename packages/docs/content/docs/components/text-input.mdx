## Installation

```bash
bunx shadcn@latest add https://raw.githubusercontent.com/cjroth/gridland/main/packages/ui/registry.json text-input
```

## Usage

```tsx title="Basic"
import { TextInput } from "@/components/ui/text-input"

<TextInput
  label="Username"
  placeholder="enter your name"
  prompt="> "
/>
```

```tsx title="Controlled with validation"
const [value, setValue] = useState("")

<TextInput
  label="Email"
  value={value}
  onChange={setValue}
  onSubmit={(v) => console.log("Submitted:", v)}
  placeholder="user@example.com"
  prompt="> "
  required
  maxLength={50}
  description="We'll never share your email"
  error={value && !value.includes("@") ? "Invalid email" : undefined}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | Field label shown above the input |
| `description` | `string` | - | Helper text shown inline next to the label |
| `error` | `string` | - | Error message (overrides description) |
| `required` | `boolean` | `false` | Shows required indicator on label |
| `disabled` | `boolean` | `false` | Disables the input |
| `value` | `string` | - | Controlled value |
| `onChange` | `(value: string) => void` | - | Called on every keystroke |
| `onSubmit` | `(value: string) => void` | - | Called on Enter key |
| `placeholder` | `string` | - | Placeholder text |
| `prompt` | `string` | - | Prompt string before the input |
| `focus` | `boolean` | `true` | Whether the input is focused |
| `maxLength` | `number` | - | Maximum input length |
