The `StatusBar` component from `@gridland/ui` renders a horizontal bar that
displays keybinding hints showing key combinations and their associated actions.

</div>

<RunDemo name="status-bar" />

## Installation

```bash
bunx shadcn@latest add https://raw.githubusercontent.com/cjroth/gridland/main/packages/ui/registry.json status-bar
```

## Usage

```tsx title="Example"
import { StatusBar } from "@/components/ui/status-bar"

<StatusBar
  items={[
    { key: "Tab", label: "switch focus" },
    { key: "q", label: "quit" },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `StatusBarItem[]` | -- | Array of keybinding hints to display |
| `extra` | `ReactNode` | `undefined` | Optional inline content rendered left of the hints (must be `<span>` or string) |

### StatusBarItem

| Field | Type | Description |
|-------|------|-------------|
| `key` | `string` | Key or key combination text |
| `label` | `string` | Description of what the key does |
