The `Ascii` component from `@gridland/ui` renders text as ASCII art using
various font styles.

<div className="my-8">

</div>

<RunDemo name="ascii" />

## Installation

```bash
bunx shadcn@latest add https://raw.githubusercontent.com/cjroth/gridland/main/packages/ui/registry.json ascii
```

## Usage

```tsx title="Example"
import { Ascii } from "@/components/ui/ascii"

<Ascii text="Hello" font="slick" color="#88c0d0" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | - | Text to render as ASCII art |
| `font` | `"tiny" \| "block" \| "slick" \| "shade"` | - | ASCII font style |
| `color` | `string` | - | Foreground color |
