## Installation

```bash
bunx shadcn@latest add https://raw.githubusercontent.com/cjroth/gridland/main/packages/ui/registry.json gradient
```

## Usage

```tsx title="Import"
import { Gradient } from "@/components/ui/gradient"
```

### Named Gradient

```tsx title="Named gradient"
<Gradient name="rainbow">Hello World!</Gradient>
```

### Custom Colors

```tsx title="Custom colors"
<Gradient colors={["#ff6b6b", "#feca57", "#48dbfb"]}>Custom gradient</Gradient>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | - | Text content to apply the gradient to |
| `name` | `GradientName` | - | Named gradient preset (see list below) |
| `colors` | `string[]` | - | Array of hex colors for custom gradient |

> **Note:** `name` and `colors` are mutually exclusive. Provide exactly one.

## Available Gradient Names

`cristal`, `teen`, `mind`, `morning`, `vice`, `passion`, `fruit`, `instagram`, `atlas`, `retro`, `summer`, `pastel`, `rainbow`
