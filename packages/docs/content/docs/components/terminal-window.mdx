The `TerminalWindow` component from `@gridland/ui` renders a decorative container that mimics
a terminal window, with traffic light buttons and an optional centered title. By default it has a dark gray background to look like a real terminal.

<div className="my-8">
  <TerminalWindowDemo />
</div>

<RunDemo name="terminal-window" />

## Installation

```bash
bunx shadcn@latest add https://raw.githubusercontent.com/cjroth/gridland/main/packages/ui/registry.json terminal-window
```

## Usage

```tsx title="Example"
import { TerminalWindow } from "@/components/ui/terminal-window"

<TerminalWindow title="my-terminal">
  <div>Window content here</div>
</TerminalWindow>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | -- | Content below the title bar |
| `className` | `string` | `undefined` | Additional CSS classes |
| `title` | `string` | `undefined` | Centered title bar text |
| `minWidth` | `number \| string` | `undefined` | Minimum width of the window |
| `transparent` | `boolean` | `false` | When true, the window background is transparent instead of dark gray |
| `onClose` | `() => void` | `undefined` | Close button callback |
| `onMinimize` | `() => void` | `undefined` | Minimize button callback |
| `onMaximize` | `() => void` | `undefined` | Maximize button callback |
